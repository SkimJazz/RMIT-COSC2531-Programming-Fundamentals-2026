# Week 10 Assessment 3 - Warehouse App | RMIT COSC2531 Programming Fundamentals

## DESCRIPTION

- Assessment 3 Object Oriented Programming Exercise
- DUE DATE: Thursday, Week 10 @ 23:59 (Melbourne time)
- TOPICS ASSESSED: (weeks 1-10)
  - Object-orientated applications
- 30% of total grade for COSC2531

This assessment follows on from your work in the two problem-solving exercises in Assessment 1. Your
task is to design and code a program that satisfies specific functional requirements that are
provided to you. You will continue to work with the program you created for Assessment 1, adapting
it into a full Parts Warehouse application. The resulting application will take into account
discounts applicable to wholesale and retail customers and will read customer and product
information from files.

You should start working on this assessment after submitting your final problem-solving exercise in
Week 8.
