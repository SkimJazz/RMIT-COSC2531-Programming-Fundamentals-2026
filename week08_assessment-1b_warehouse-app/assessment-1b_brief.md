In the course content in Week 7 (Activity 7.3) you will refactor ABC Bicycle’s program using the
object oriented paradigm, including the creation of Customer and Product. In Assessment 1B, you will
build on the program created in Activity 7.3. You will rename the Product class into a Part class,
and create a class for assembled part (AssembledPart) which is a subclass of Part. An assembled part
is assembled from exactly two standard parts.
